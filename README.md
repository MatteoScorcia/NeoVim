## Required installation

- Neovim installation

  ```
  brew install neovim
  ```
- Config files setup (MacOS)

  ```
  mkdir ~/.config/nvim
  cd ~/.config/nvim
  git clone https://github.com/MatteoScorcia/NeoVim.git
  ```

- Neovim python support

  ```
  pip3 install pynvim
  ```

- Neovim node support

  ```
  npm i -g neovim
  ```
